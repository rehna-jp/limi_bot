# Limi — Branding Brief

## Logo

**Concept:** Lowercase `l` inside a rounded square. The letterform reads simultaneously as a letter and a single candlestick bar — one vertical stroke with a short horizontal tick at the top.

**Background:** `#0A0A0A` (near-black)

**Accent:** `#C6FF3D` (lime green — high contrast against dark background, readable at small sizes)

**Typeface:** Geometric sans-serif. The `l` should be rendered in the accent colour, heavy weight. No serifs. Similar in feel to Inter Bold or Satoshi Black, but the `l` should be custom-drawn so the proportions read as a candlestick shape, not just a letterform.

**Shape:** Rounded square container, corner radius approximately 22–24% of width. Same proportions as a Telegram avatar (512×512 source, displayed at 40×40). Test it at 40px — it should still read clearly at that size.

**Don't do:**
- Don't add a chart line or additional candles — one bar is enough
- Don't add text below the icon in the avatar version
- Don't use gradients — flat colour only

## Wordmark

`limi` in lowercase, geometric sans-serif, same typeface as the logo `l`. Set in white or `#C6FF3D` on dark backgrounds. Letter-spacing: -0.5 to -1%.

## Colour palette

| Name | Hex | Usage |
|---|---|---|
| Background | `#0A0A0A` | Primary bg, message cards |
| Lime | `#C6FF3D` | Primary accent, CTA, highlight |
| White | `#FFFFFF` | Body text |
| Muted | `#888888` | Secondary text, metadata |
| Green | `#22C55E` | Positive PnL, YES odds |
| Red | `#EF4444` | Negative PnL, NO odds when dominant |

## Tone

Sharp. Brief. No marketing fluff. The copy standard for Limi is: if it sounds like a chatbot wrote it, rewrite it. Every string should read like a trader texted it to another trader.

Examples:
- ✅ `64% YES — $1.2M vol`
- ❌ `The market currently shows a 64% probability for a YES outcome with $1.2 million in 24-hour trading volume`
- ✅ `Watch set.`
- ❌ `Your alert has been successfully configured!`
